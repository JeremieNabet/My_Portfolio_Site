# Generic Showcase Template

Reusable one-page website starter for service businesses, freelancers, agencies, restaurants, and local brands.

## Included

- `index.html`: semantic page structure
- `styles.css`: design tokens + responsive layout
- `script.js`: mobile menu toggle + smooth anchors + form demo behavior

## How to reuse

1. Copy the `template-site-vitrine` folder wherever you want.
2. Replace brand name, texts, colors, and links in `index.html`.
3. Customize colors in `:root` in `styles.css`.
4. Add your own images in an `assets/` folder and update paths.
5. Connect the contact form to your backend endpoint.

## Notes

- The form currently shows a success message only on the front-end.
- No framework required.
- Works as a static website.
